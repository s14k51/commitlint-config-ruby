# commitlint-config-ruby

Opinionated `commitlint` config. Follows [conventional commits spec](https://conventionalcommits.org/).

# Installation

`npm i -D @commitlint/cli commitlint-config-ruby`

# Usage

```js
// commitlint.config.js
module.exports = {
  extends: ['ruby']
}
```
